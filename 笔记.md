## 💻 常用终端命令

### 本地开发
```bash
cd mygardens          # 进入项目文件夹
npm start             # 在本地预览网站（打开 http://localhost:3000）
# 按 Control + C 停止本地预览
```

### 部署到网上
```bash
vercel                # 部署网站（第一次用，会让你登录）
vercel --prod         # 之后更新网站用这个
```

## 📚 今天学到的概念

| 名词 | 解释 |
|---|---|
| **终端 Terminal** | 用文字命令控制电脑的黑色窗口 |
| **npm** | 程序员的"App Store"，用来下载各种工具包（全球200万+工具免费用） |
| **localhost:3000** | 只有自己电脑能看到的本地预览地址 |
| **React** | 让网页能动起来的框架，MyGardens 就是用它写的 |
| **Vercel** | 免费的网站托管平台，把代码放上去就能让全世界访问 |
| **npm start** | 在自己电脑上"试菜" |
| **vercel** | 把菜"上架到外卖平台"，全世界都能点 |

## 🛠 今天做了什么（步骤回顾）

1. 安装了 Node.js
2. 在终端运行 `npx create-react-app mygardens` 创建项目
3. 把 MyGardens.jsx 的代码替换进 `src/App.js`
4. 运行 `npm start` 本地预览成功 ✅
5. 安装 Vercel 工具：`npm install -g vercel`
6. 运行 `vercel` 登录并部署
7. 网站成功上线 🎉
